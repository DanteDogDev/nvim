## Installation

### Commands
```bash
Remove-Item $env:LOCALAPPDATA\nvim -Recurse -Force
Remove-Item $env:LOCALAPPDATA\nvim-data -Recurse -Force
git clone https://github.com/DanteDogDev/nvim.git $env:LOCALAPPDATA\nvim
```
